01. 요소 노드의 텍스트 조작

nodeValue

        nodeValue (property)
        - getter, setter가 모두 존재하는 프로퍼티
        - nodeValuev 프로퍼티 참조 시 노드 객체의 값(텍스트 노드의 텍스트)을 반환해준다.
        - 주의 : 텍스트가 아닌 요소 노드는 null을 반환한다.
변경 전

textContent

        textContent 프로퍼티도 setter 와 getter 가 모두 존재하는 프로퍼티다.
        하지만 위에서 nodeValue와 다른 점은 요소 노드의 텍스트와 모든 자손 노드의 텍스트를 모두 췯그하거나 변경할 수 있다.
